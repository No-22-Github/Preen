**核心后端(训练基于此)**
- https://github.com/ml-explore/mlx-lm — 关键文件 `mlx_lm/models/rwkv7.py`。RWKV-7 完整前向,Apple 官方维护。训练需 patch `Rwkv7TimeMixing._wkv7` 强制走 `_wkv7_step_ops`(纯 ops、可微);默认的 `mx.fast.metal_kernel` 路径无 VJP,梯度会静默断裂。state 注入点为每层 cache 的 `[1]` 位,形状 (B, H, D, D)

**模型权重**
- RWKV 官方已停止更新 World 数据系模型,现行主力为 G1x 系列(本项目 0.4B 使用 g1d)。fla-hub 上虽有 HF 格式的 g1 仓库,但转换自 2025 年初代 G1,已落后数代,**禁止直接使用**。正确做法:将 BlinkDL 原生 `.pth` 用官方脚本转换为 HF 格式后再经 `mlx_lm.load` 加载,参考脚本:`https://github.com/fla-org/flash-linear-attention/blob/main/utils/convert_from_rwkv7.py`。转换注意两点:① 使用 `--precision bf16`(默认 fp32 体积翻倍无收益);② 脚本输出不含 tokenizer,需从 `fla-hub/rwkv7-0.1B-g1` 拷贝 `hf_rwkv_tokenizer.py`、`tokenizer_config.json`、`added_tokens.json` 至输出目录(World tokenizer 全系通用)。转换完成后先用 `mlx_lm.load` 加载并生成一句文本验证正确性,再进入训练流程。
